SAPERION Render Service documentation

/renderservice/webapi/version

Displays the version of the service.

Produces: text/plain

Method: GET

/renderservice/webapi/formats/sources

Returns the supported source-formats as array of JSON representations of com.saperion.connector.formats.serializable.SerializableSourceFormat.

Produces: application/json

Method: GET

/renderservice/webapi/formats/targets

Returns the supported target-formats as array of JSON representations of com.saperion.connector.formats.serializable.SerializableTargetFormat.

Produces: application/json

Method: GET

/renderservice/webapi/rendition/<format>

Renders to the specified format (e.g. 'pdf' or 'png').

Accepts: multipart/form-data

Produces: depends on the specified format

Method: POST

Parameter Type Explanation Optional Default
timeout int Max time to wait for rendition yes 10 (configurable)
dpi int Resolution of rendition yes 200 (configurable)
page int Page to render yes 1
height int Height of rendition in pixels. yes -1 (auto)
width int Width of rendition in pixels yes -1 (auto)
file file Content to render no

Returncodes in case of an error

400

  • Request contained an invalid parameter.

413

  • Returned when the document posted in the request is larger than the configured maximum size.

500

  • Render thread was aborted with an exception.
  • I/O error when rendering.
  • Rendering failed because of an unsupported document or another error in the engine.
  • Timeout when waiting for the result of a render-thread.
  • Generic internal errors.
